## 3) Customization options (allowed substitutions)

These are *allowed* swaps; anything else routes to escalation.

- **Allowed substitutions**:
  - Breakfast protein: bacon ↔ pork bacon ↔ turkey bacon
  - Breakfast sandwich bread: croissant ↔ plain bagel ↔ everything bagel
  - Sandwich: red onion optional (include or omit)
  - Spring mix salad: bacon bits add-on optional (include or omit)
- **Not allowed**:
  - Any item not listed in this file
  - Any “new hot dish” not listed under Breakfast/Lunch

## 4) Dietary accommodations (supported)

Keep this brutally honest. “Supported” means repeatable at scale.

- **Vegetarian**: Limited (e.g., grits, fruit cup, yogurt parfait, some pastries/sides; confirm per order)
- **Vegan**: Limited (fruit cup; other items require confirmation)
- **Gluten-free**: Limited (requires confirmation; avoid bread/pasta/pastries)
- **Nut-free**: Possible but not guaranteed (nuts appear in charcuterie; cross-contact risk—confirm)
- **Halal/Kosher**: Not guaranteed (confirm per request)
- **Other**: Escalate if the request requires ingredient or process changes
